# Fintech Trendwatcher

Сервис для поиска финтех-сигналов: парсер собирает новости, модель превращает их в карточки, backend дает доступ к данным.

## Запуск

```powershell
python back\init_db.py # обнуление текущей БД
python back\app.py
```

Backend поднимается на:

```text
http://127.0.0.1:5000
```

Запустить парсер по всем активным источникам из таблицы `sources`:

```text
http://127.0.0.1:5000/parser
```

## Источники

Для пополнения списка источников используется консольный скрипт:

```powershell
python back\update_sources.py
```

Он спрашивает параметры источника и парсера, делает тестовый запрос, показывает `Found: N`, затем спрашивает, сохранять источник в БД или нет.

## Тесты

Новые тесты в `tests/`, файлы должны называться `test_*.py`.

Запуск:

```text
http://127.0.0.1:5000/tests/all
http://127.0.0.1:5000/tests/db
```

Тесты также запускаются:

- на `push` и `pull_request` через GitHub Actions;
- перед `commit`, `push` и `merge` через локальные git hooks из `.githooks`.
